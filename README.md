#  Access Key Generator

A Flutter-based mobile application that generates secure access keys using a custom Device ID and Instance ID logic. The app features a modern red-and-white UI design with stylish typography and responsive layout.

---

## Features

- Device ID input (1–16 digits)
- Instance ID input (1–20 range)
- Input validation with error dialogs
- Clean and responsive UI
- Google Fonts integration
- Custom logo integration

---

##  Technologies Used

- Dart
- Material Design

##  Getting Started

### Prerequisites

- Flutter SDK installed
- Android Studio or VS Code
- Android SDK / Emulator or physical device

Check setup:

```
flutter doctor
```

---

### Installation

1. Clone the repository:

```
git clone https://github.com/your-username/mentor-circuits.git
```

2. Navigate into the project:

```
cd mentor-circuits
```

3. Install dependencies:

```
flutter pub get
```

4. Run the app:

```
flutter run
```

---

##  Build Release APK

```
flutter build apk --release
```

The APK file will be generated at:

```
build/app/outputs/flutter-apk/app-release.apk
```
